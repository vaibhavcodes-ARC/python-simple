# DEV PALACE – Hotel Management System
A Python Project for Class 12 CBSE Board Exam

## Project Overview
DEV PALACE is a console-based Hotel Management System built using Python and CSV file handling.

## Features
- Room Booking
- Food Ordering
- Arcade Games
- Massage Services
- Vehicle Rentals
- Complaints & Feedback
- Staff Dashboard

The system provides two modes of access:

Customer Mode – Allows hotel guests to use services

Staff Mode – Allows hotel staff to manage backend operations

Everything is stored in CSV files, without any external database or GUI.

## How to Run
python "Dev-Hotel.py"
